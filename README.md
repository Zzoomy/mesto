# Проект: Место

### Обзор
---

 Адаптивный одностраничный сайт, для размещения фотографиий. 
 Реализованые возможности: ввод свего имени и информации о себе, добаление новых фотографий, лайк и удаление.

---

**Используемые технологии**

* HTML
* CSS
* JavaScript
